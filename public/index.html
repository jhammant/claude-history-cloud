Features

Hybrid Search

BM25 + TF-IDF with Reciprocal Rank Fusion. Filter by project, date, tool usage. Find anything in your history.

Knowledge Extraction

Automatically pulls out decisions, solutions, and error-to-fix patterns from every session.

Session-Start Hook

Auto-injects relevant project context when you start a new Claude Code session. No manual searching.

Learnings System

Clusters similar solutions across projects. High-scoring patterns become "learnings" that surface automatically.

Team Knowledge

Share extracted patterns with your team. When someone solves a problem, everyone's Claude knows about it.

Privacy First

100% local by default. No network calls unless you enable cloud sync. E2E encryption for team data.
